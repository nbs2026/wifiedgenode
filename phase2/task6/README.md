# ESP32 IoT Irrigation System

## Objective
Control irrigation system using IoT lab API commands.

## Hardware
- ESP32 Dev Module
- Relay Module

## Working
- Device connects to WiFi
- Receives JSON commands from IoT platform
- Executes relay actions:
  - IRRIGATE → Relay ON
  - STOP → Relay OFF
